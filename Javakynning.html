Java

Velkomin í örnámskeið í forritun. Við munum fara yfir nokkra grunnhluti í Java í dag og síðan munið þið fá að sjá hvernig auðveldur leikur er framkvæmdur.

Sækja DrJava

Byrjið á því að sækja forritið DrJava, sem er mjög notandavænn Java ritill. Þið sækið hann hér. Linkurinn fer með ykkur á heimasíðu DrJava, ekki hafa áhyggjur þrátt fyrir að síðan virki ekki vinaleg sleppið því bara að smella á auglýsingarnar. ;) Ef einhver á í erfiðleikum með að sækja forritið smellið þá hér.

Núna getið þið opnað DrJava.
